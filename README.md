# 🚀 Meu Primeiro Passo com React: Formulário de Cadastro

Olá! Seja muito bem-vindo ao repositório do meu primeiríssimo projeto utilizando **React** (criado com Vite). 

Este é um projeto simples, mas que representa o início da minha jornada no desenvolvimento front-end. Ele foi construído com muito carinho para colocar em prática conceitos iniciais de componentes, estados e integração com serviços externos.

[Veja o Site](https://mellifluous-stardust-64b628.netlify.app/)

---



### 💡 Sobre o Projeto

O objetivo principal foi criar uma interface amigável de "Criação de Conta". Como eu queria ver a "mágica" acontecer de verdade, integrei o formulário a uma **API de Webhook**.

**Como funciona?**
Ao preencher os campos de Nome, Sobrenome, Email e Senha, os dados são capturados e enviados para uma URL de Webhook. Isso me permitiu simular o comportamento de um servidor real recebendo as informações em tempo real.
